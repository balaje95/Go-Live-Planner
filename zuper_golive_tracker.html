🔧 One-time Firebase Setup

This app stores all data in Firebase so every PM can sign in from any device. You only need to do this once. It takes about 3 minutes.

1
Create a free Firebase project
Go to console.firebase.google.com → click "Add project" → give it any name → continue until done.
2
Enable Realtime Database
In your project sidebar → Build → Realtime Database → Create database → choose any region → start in test mode (you can lock it down later).
3
Copy your database URL
After creating the database, copy the URL shown at the top. It looks like:
https://your-project-default-rtdb.firebaseio.com

Paste your Firebase details below

Find API Key & Project ID in Firebase Console → Project Settings → General → Your apps → Web app (create one if needed).

Please fill in all three fields.
